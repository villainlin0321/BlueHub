# 企业端我的页 Figma 切图精修设计说明

## 背景

本次改造针对 `lib/features/me/presentation/role_pages/company_me_page.dart` 的企业端“我的”页面。
用户要求基于 Figma 切图进行局部视觉精修，重点解决以下问题：

- 头部背景图颜色与设计稿不一致，需改为使用 Figma 导出的背景切图。
- 企业名称与“行业/地址”之间的垂直间距需减少 `2px`，且资料区与下方统计卡的间距需加大。
- 统计卡数字字号需加大，数字与下方标签的间距需加大。
- “应聘管理”菜单图标当前视觉上偏大，需单独校正到与其他菜单图标一致。

## 范围

- 仅修改 `company_me_page.dart` 中头部与菜单视觉相关实现。
- 允许接入 `.figma/image` 中已导出的背景与图标资源。
- 不修改页面跳转逻辑、数据来源、状态管理和菜单业务行为。

## 设计方案

### 1. 背景切图

- 将头部蓝色背景从代码渐变切换为 Figma 导出的背景资源 `mrc7dh3e-2p5yozt.svg`。
- 保留头部整体高度口径 `220 = 44 安全区 + 176 内容区`，只替换背景表现方式，不改变交互逻辑。

### 2. 资料区间距

- 企业名称与“行业/地址”文案的垂直间距由当前值减少 `2px`。
- 资料区整体与统计卡之间的距离适当拉大，以更贴近 Figma 中头像信息区与卡片的留白关系。

### 3. 统计卡字重与间距

- 调大统计数字字号，使其更接近设计稿中的视觉重心。
- 增加数字与下方标签之间的垂直距离，增强上下层级。
- 不改动统计项数量和数据来源。

### 4. 菜单图标一致性

- 单独修正“应聘管理”图标的渲染尺寸，使其与“企业资质 / 人才中心 / 签证服务 / 订单管理”保持一致。
- 优先通过局部图标尺寸参数处理，避免影响其他菜单项。

## 实施方式

- 在 `company_me_page.dart` 中优先替换头部背景层为 `SvgPicture.asset(...)`。
- 根据 Figma 标注同步微调资料区和统计卡的间距、字号与定位。
- 对关键布局调整补充中文注释，保持实现可读性。

## 验收标准

- 头部背景视觉与 Figma 切图一致，不再出现代码渐变造成的色差。
- 企业名称与副信息之间的间距减少，且资料区与统计卡间距更接近设计稿。
- 统计数字更大，数字与标签之间的间距更合理。
- “应聘管理”图标大小与其他菜单图标一致。
- 页面无新的布局报错，现有跳转与数据展示逻辑保持正常。
